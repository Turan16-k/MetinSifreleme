# 75 — Metin Şifreleme

Caesar, Vigenère ve AES ile metin şifreleyen Flask aracı.

## ▶️ Çalıştır
```bash
pip install -r requirements.txt
python app.py   # http://localhost:5000
```

## ✨ Özellikler
- Caesar / Vigenère (klasik)
- AES-Fernet (modern)
- Şifrele/çöz arayüzü

## 🧠 Notlar
Klasik şifrelerle modern simetrik şifreleme (AES) arasındaki güvenlik farkını kavradım.
